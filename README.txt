This web app uses node.js, express, and the OMDB API to allow users to search
for movies and save them to a favorites list.

To run this app, install all of the node modules specified in the package.json
file. Then type node app to start the web server.

Note that this app requires a connection to a specifically configured server.
You must run the config.sql file on your PostgreSQL server to get your database
ready for hosting this web app.
